Fourty is built to be run by you, on your own Postgres, with nothing else to operate.

- **[Installation](./installation.md)** — Docker Compose (recommended) or from source.
- **[Configuration](./configuration.md)** — the complete environment-variable reference.
- **[Upgrading & migrations](./upgrading.md)** — reversible schema migrations, and
  importing from SQLite or Twenty.
- **[Operations](./operations.md)** — backups, observability, rate limits, and the
  security posture.
